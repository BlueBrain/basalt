basalt.cli module¶

basalt-cli - CLI utility to deal with a basalt connectivity graph

Usage:
basalt-cli ngv import neuroglial [–max-astrocytes=<nb>] [–create-nodes] <h5-file> <basalt-path> basalt-cli ngv import synaptic [–max-neurons=<nb>] [–create-nodes] <h5-file> <basalt-path> basalt-cli ngv import gliovascular [–max-astrocytes=<nb>] [–create-nodes] <h5-connectivity> <h5-data> <basalt-path> basalt-cli -h | –help basalt-cli –version
Options
--max-astrocytes=<nb>
 Maximum number of astrocytes to import [default: -1].

-h –help Show this screen. –version Show version.
