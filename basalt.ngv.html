basalt.ngv.importer.import_microdomain(h5_file, basalt_path, max_=-1, create_vertices=False)[source]¶

Import microdomain structure in HDF5 file in a basalt graph

Import microdomains subgraph and attach every microdomain to its associated astrocyte.

Parameters:
  • h5_file – path to HDF5 file
  • basalt_path – path to basalt graph
  • max – limit number of microdomains to import. If negative, then microdomains of every astrocytes are imported.
  • create_vertices – if True then also creates astrocyte vertices connected to the microdomains
Returns:

A dict providing import information and statistics
